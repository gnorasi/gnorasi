Συνοπτικός οδηγός εγκατάστασης του ΓΝΩΡΑΣΗ από το αποθετήριο πηγαίου κώδικα της εφαρμογής

Η εγκατάσταση πραγματοποιήθηκε σε Windows 7 Professional Service Pack 1 64-bit.

Προαπαιτούμενα:

	1. Windows 7.

	2. Visual Studio Express Edition (http://www.microsoft.com/visualstudio)

		Εγκατάσταση visual studio express edition από το παρακάτω link :
		http://www.microsoft.com/visualstudio/eng/downloads#d-2010-express
		Η εγκατάσταση γίνεται μέσω wizard.
		Με την εγκατάσταση Visual Studio Express 2010 δεν έρχεται ένας debugger, έτσι πρέπει να γίνει ξεχωριστά 
		η εγκατάσταση του.
		Κατέβασμα του εργαλείου Windows Software Development Kit για Windows από το παρακάτω link
		http://msdn.microsoft.com/en-us/windows/hardware/gg463009
		Για το debugging χρειαζόμαστε μόνο τα Debugging Tools από το Kit, έτσι στον installation wizzard μόνο τα Debugging Tools.

	3. cmake (http://www.cmake.org/) 

		Κατέβασμα του αρχείου εγκατάστασης από το παρακάτω link:
		http://www.cmake.org/files/v2.8/cmake-2.8.10-win32-x86.exe .
		Εγκατάσταση από τον wizard. 
		Στις παραμέτρους εγκατάστασης τσεκάρουμε την επιλογή να προσθέσει το cmake στις μεταβλητές συστήματος (Add to PATH).

	4. Εγκατάσταση Qt (http://qt-project.org/downloads) 

		Κατέβασμα του αρχείου εγκατάστασης από το παρακάτω link :
		http://releases.qt-project.org/qt4/source/qt-win-opensource-4.8.3-vs2010.exe .
		Straight forward εγκατάσταση μέσω wizzard.

	5. Εγκατάσταση Qt Creator IDE (http://qt-project.org/downloads) 

		Κατέβασμα του αρχείου εγκατάστασης από το παρακάτω link:
		http://origin.releases.qt-project.org/qtcreator/2.6.0/qt-creator-windows-opensource-2.6.0.exe .
		Η εγκατάσταση γίνεται μέσω wizard.
			****************************************************************************************************************
			Αν δεν έχει βρεθεί αυτόματα η version της Qt, ρυθμίζουμε την χρήση των βιβλιοθηκών της Qt. 
			Tools – Options, στο δίαλογο που ανοίγει επιλέγουμε Build & Run στην αριστερή στήλη και το tap Qt Versions. 
			Στο tab Qt Versions κάνουμε Add και επιλέγουμε την διαδρομή για το αρχείο qmake.
			****************************************************************************************************************
		Για τη χρήση multi-core CPU με τον Qt Creator, μπορείτε να ενεργοποιήσετε το jom από Tools - Build & Run - General
		τσεκάρουμε την επιλογή :
			Use jom insteal of nmake


Build and run του Gnorasis

Build
 
	Μέσα στο QtCreator:
		File / Open File or Project και επιλέγετε το "path/to/gnorasi/Gnorasi/CMakeLists.txt"
		Επιλέγουμε το build directory και τρέχουμε το cmake.
		Τα cmake arguments για debug και release αντίστοιχα : 
			-DCMAKE_BUILD_TYPE=Debug 
			-DCMAKE_BUILD_TYPE=Release
		
		
Run
	Run
	

Debug
	Μέσα από το QtCreator Tools - Build & Run - Kits - Desktop , επιλέγουμε τον debugger που εγκαταστάθηκε από το 
	Debugging Tools.
